🏗️ Što su klase i objekti?

🍪 Analogija s kolačićima:

Klasa = Kalup za kolačiće (blueprint, recept)
Objekt = Pojedini kolačić napravljen tim kalupom

Jedan kalup → Bezbroj kolačića! 🎉
  • Klase definiraju vlastite tipove podataka
  • Atributi (varijable) = karakteristike entiteta
  • Metode (funkcije) = ponašanje entiteta
  • Mogu se višestruko koristiti nasljeđivanjem

💳 Primjer: Klasa Account

package account;

public class Account {
private String name; // atribut (varijabla instance)

// Getter metoda - dohvaća vrijednost
public String getName() {
return name;
}
// Setter metoda - postavlja vrijednost
public void setName(String name) {
this.name = name; // 'this' razlikuje parametar od varijable
}
}
💡 Tip: U IntelliJ-u koristi Alt+Insert za automatsko generiranje getter/setter metoda!

📝 Konvencija nazivanja (Naming Convention)

🐫
  • Klase: PascalCase → velika početno slovo
  • Objekti i varijable: camelCase → malo početno slovo
  • Metode: camelCase → malo početno slovo
  • Svaka klasa mora biti u istoimenu .java datoteku
  • ❌ Izbjegavati hrvatske znakove (č, ć, š, ž, đ)

🎨 Kreiranje objekta

import java.util.Scanner;
public class AccountTest {
public static void main(String[] args) {
// Kreiranje Scanner objekta za unos
Scanner unos = new Scanner(System.in);
// Kreiranje Account objekta pomoću konstruktora
Account mojRacun = new Account();
System.out.println("Unesite naziv računa: ");
String brojRacuna = unos.nextLine();
mojRacun.setName(brojRacuna); // Postavljanje naziva
unos.close();
}
}
🔑 new = ključna riječ za kreiranje novog objekta (poziva konstruktor)

⚡ Primitivni vs Referentni tipovi

Primitivni tipovi

  • int, long, short, byte
  • float, double
  • boolean
  • char
📊 Defaultna vrijednost:
0 ili false

Referentni tipovi

  • String
  • Account
  • Scanner
  • Sve klase...
📊 Defaultna vrijednost:
null
⚠️ NullPointerException: Događa se kad pokušaš koristiti metodu ili varijablu objekta koji je null!

🏗️ Konstruktori

🏭 Konstruktor = Tvornica za objekte!
Specijalna metoda koja "sastavlja" objekt i inicijalizira njegove vrijednosti
public class Account {
private String name;
// Konstruktor - isto ime kao klasa, nema return type!
public Account(String name) {
this.name = name;
}
}
// Korištenje:
Account racun = new Account("Tekući račun");
💡 Ako ne napišeš konstruktor, Java automatski kreira defaultni konstruktor (bez parametara)
🎯 IntelliJ: Code → Generate... → Constructor

📅 Date and Time API (Java 8+)

🎉 Java 8 donosi novi Date/Time API inspiriran JodaTime bibliotekom!
Stari Date i Calendar su zastarjeli i konfuzni

Glavne klase:

  • LocalDate - samo datum (godina, mjesec, dan)
  • LocalTime - samo vrijeme (sati, minute, sekunde)
  • LocalDateTime - datum + vrijeme
  • Instant - timestamp (nanosekunde od 01.01.1970.)

⏰ Primjer: LocalDateTime

import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
LocalDateTime trenutno = LocalDateTime.now();
System.out.println("Trenutni datum: " + trenutno);
// Dodavanje vremena
LocalDateTime zaKvaTjedna = trenutno.plusWeeks(2);
System.out.println("Nakon dva tjedna: " + zaKvaTjedna);
// Formatiranje
String formatiran = trenutno.format(
DateTimeFormatter.ofPattern("dd.MM.yyyy.") );
System.out.println("Formatirani: " + formatiran);
// Output:
Trenutni datum: 2025-10-10T12:37:29.035 Nakon dva tjedna: 2025-10-24T12:37:29.035 Formatirani: 10.10.2025.

secondDate = firstDate; // Kopiranje reference!

Obje varijable pokazuju na isti objekt!

⚠️ Promjena preko jedne varijable mijenja objekt za obje! Dijele memorijsku lokaciju!

⚖️ Uspoređivanje objekata

Dva načina usporedbe:

== (usporedba referenci)

if (firstDate == secondDate) { // Jesu li isti objekti?
// (ista memorijska lokacija)
}
⚠️ Često se koristi pogrešno!

.equals() (usporedba vrijednosti)

boolean areDatesEqual(Date f, Date s) { return f.day == s.day && f.month.equals(s.month) && f.year == s.year; }
✅ Pravilan način!
💡 Za String UVIJEK koristi .equals(), ne ==!
